# WDI Sustainability Dashboard

An interactive Streamlit dashboard exploring exchange rate volatility 
and sustainable development indicators across fragile and developing states.

## Data Source
World Development Indicators (WDI), World Bank (2026)

## Indicators
- `PA.NUS.FCRF` — Official exchange rate (LCU per US$)
- `NY.GDP.PCAP.CD` — GDP per capita (current US$)
- `SI.POV.DDAY` — Poverty headcount ratio at $2.15/day
